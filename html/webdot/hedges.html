Dot supports a "\N" short form so that the URLs of the nodes can be based on the node name. Edges can also have URLs, but because of the limited abilities of image maps, a label must exist and the URL is attached to it. 
digraph G {
	graph [URL="default.html"]
	node [URL="\N.html"]
	a -> b -> c
	a -> c [URL="a->c.html", label="a->c"]
}

The box around the clickable image can be suppressed with "border=0"
